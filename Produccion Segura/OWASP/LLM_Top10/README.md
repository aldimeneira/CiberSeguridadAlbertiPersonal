# Ejemplos reales de vulnerabilidades del Top 10 de LLM

Este directorio contiene dos ejemplos prácticos de vulnerabilidades de aplicaciones con LLM, cada uno con:

- `vulnerabilidad.md`: análisis de la debilidad y su impacto
- `demostracion.md`: ejemplo de explotación en entorno controlado
- `mitigacion.md`: medidas de mitigación y buenas prácticas

## Ejemplos incluidos

1. Inyección de prompt en agentes con herramientas
2. Divulgación de información sensible en aplicaciones con contexto de usuario
